OptimalUncertaintyQuantification.jl: Optimal Uncertainty Quantification

DISTRIBUTION STATEMENT A. Approved for public release: distribution unlimited. Case Number: AFRL-2024-5455. Cleared 10/2/2024.

The Optimal Uncertainty Quantification (OUQ) algorithm provides a means of computing the bounds of the expectations of quantities of interest despite not having complete knowledge of the probability distribution of the uncertain variables. This is achieved by finding the worst/best case distributions in some set $\mathcal{A}$ of possible distributions given the knowledge available.

OptimalUncertaintyQuantification.jl implements the OUQ algorithm and its convex and "moment class" forms in the Julia programming language, using techniques based on complete and rigorous global methods in order to bound the effects of finite computation on the OUQ bounds.

Installation

To install OptimalUncertaintyQuantification.jl, use the Julia package manager:

using Pkg
Pkg.add("OptimalUncertaintyQuantification")
